## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 8 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya mengembangkan **Notes App** dengan menambahkan platform features, yaitu dependency injection menggunakan Koin, device info, dan network status indicator.

## Fitur Aplikasi

1. Setup Koin Dependency Injection untuk seluruh aplikasi.
2. Implementasi `DeviceInfo` menggunakan `expect/actual`.
3. Implementasi `NetworkMonitor` menggunakan `expect/actual`.
4. Menampilkan device info pada Settings Screen.
5. Menampilkan network status indicator pada Main Screen.
6. Semua dependencies di-inject melalui Koin.

## Teknologi yang Digunakan

- Kotlin
- Jetpack Compose
- Kotlin Multiplatform
- Koin Dependency Injection
- expect/actual
- Android Studio

## Cara Menjalankan Aplikasi

1. Clone repository berikut:

   ```bash
   git clone https://github.com/14-169-RaisyaSyifaSaleh/TugasPAM_123140169.git
   ```
2. Buka project menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi pada Android Emulator atau perangkat Android.

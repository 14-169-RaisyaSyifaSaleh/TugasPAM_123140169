## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 10 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya mengembangkan **Notes App** dengan menerapkan Dependency Injection menggunakan Koin serta menambahkan pengujian pada beberapa bagian aplikasi, seperti unit test untuk repository, ViewModel, flow test, dan UI test.

## Fitur yang Dikerjakan

1. Setup Koin Dependency Injection dengan minimal 2 module:
   - Data module
   - ViewModel module

2. Unit test untuk `NoteRepository` dengan minimal 5 test cases.

3. Unit test untuk `NotesViewModel` menggunakan MockK dengan minimal 4 test cases.

4. Flow test menggunakan Turbine dengan minimal 2 test cases.

5. UI test untuk `NotesScreen` dengan minimal 3 test cases.

6. Minimum code coverage 60% untuk business logic.

## Teknologi yang Digunakan

- Kotlin
- Jetpack Compose
- Koin Dependency Injection
- MockK
- Turbine
- JUnit
- Android Studio

## Cara Menjalankan Aplikasi

1. Clone repository berikut:

   ```bash
   git clone https://github.com/14-169-RaisyaSyifaSaleh/TugasPAM_123140169.git
   ```
2. Buka project menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi pada Android Emulator atau perangkat Android.

## Cara Menjalankan Test
``` bash
./gradlew test
```

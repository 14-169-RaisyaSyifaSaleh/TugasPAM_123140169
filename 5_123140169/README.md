## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 5 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya mengembangkan **Notes App** dari tugas minggu sebelumnya dengan menambahkan fitur navigasi menggunakan bottom navigation, note list, floating action button, dan perpindahan halaman antar screen.

## Fitur Aplikasi

1. Bottom Navigation dengan 3 tab:
   - Notes
   - Favorites
   - Profile

2. Note List yang dapat menampilkan daftar catatan dengan passing `noteId`.

3. Floating Action Button untuk menambahkan note baru dan berpindah ke `AddNoteScreen`.

4. Back navigation yang berjalan dengan benar pada setiap screen.

5. Edit Note Screen dengan passing `noteId` sebagai argument.

1. Clone repository berikut:
```
git clone https://github.com/14-169-RaisyaSyifaSaleh/TugasPAM_123140169.git
```
2. Buka project menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi pada Android Emulator atau perangkat Android.

## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 6 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya membuat aplikasi **News Reader** yang dapat mengambil data berita dari public API, menampilkan daftar artikel, membuka detail artikel, serta menerapkan fitur pull to refresh dan state handling.

## Fitur Aplikasi

1. Fetch berita dari public API.
2. Menampilkan list artikel berisi title, description, dan image.
3. Detail screen saat artikel diklik.
4. Pull to refresh untuk memperbarui data berita.
5. Menampilkan state loading, success, dan error.
6. Repository pattern untuk mengatur API calls.

## Teknologi yang Digunakan

- Kotlin
- Jetpack Compose
- Retrofit / Ktor
- Coroutines
- ViewModel
- Repository Pattern
- Android Studio

## Cara Menjalankan Aplikasi

1. Clone repository berikut:

   ```bash
   git clone https://github.com/14-169-RaisyaSyifaSaleh/TugasPAM_123140169.git
   ```
2. Buka project menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi pada Android Emulator atau perangkat Android.

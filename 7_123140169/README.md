## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 7 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya mengembangkan **Notes App** dengan menambahkan fitur penyimpanan data lokal menggunakan SQLDelight, operasi CRUD, fitur pencarian catatan, pengaturan aplikasi, serta dukungan offline-first.

## Fitur Aplikasi

1. SQLDelight database untuk menyimpan notes.
2. CRUD operations:
   - Create note
   - Read note
   - Update note
   - Delete note

3. Search functionality untuk mencari notes.
4. Settings screen dengan DataStore untuk menyimpan:
   - Theme
   - Sort order

5. Offline-first, sehingga data tersimpan secara lokal.
6. UI states yang sesuai:
   - Loading
   - Empty
   - Content

## Teknologi yang Digunakan

- Kotlin
- Jetpack Compose
- SQLDelight
- DataStore
- Coroutines
- ViewModel
- Android Studio

## Cara Menjalankan Aplikasi

1. Clone repository berikut:

   ```bash
   git clone https://github.com/14-169-RaisyaSyifaSaleh/TugasPAM_123140169.git
   ```
2. Buka project menggunakan Android Studio.
3. Jalankan aplikasi pada Android Emulator atau perangkat Android.

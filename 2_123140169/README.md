## Deskripsi Tugas

Repository ini dibuat untuk memenuhi tugas praktikum mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya membuat aplikasi **News Feed Simulator** menggunakan Kotlin. Aplikasi ini dibuat untuk mensimulasikan data berita baru setiap 2 detik, memfilter berita berdasarkan kategori, mengubah data berita ke format yang ditampilkan, menyimpan jumlah berita yang sudah dibaca, serta mengambil detail berita secara asynchronous menggunakan coroutines.

## Fitur Aplikasi

Fitur yang dibuat pada aplikasi ini adalah:

1. Flow untuk mensimulasikan data berita baru setiap 2 detik.
2. Filter berita berdasarkan kategori tertentu.
3. Transform data berita menjadi format yang siap ditampilkan.
4. StateFlow untuk menyimpan jumlah berita yang sudah dibaca.
5. Coroutines untuk mengambil detail berita secara async.

## Teknologi yang Digunakan

- Kotlin
- Flow
- StateFlow
- Coroutines

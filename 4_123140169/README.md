## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 4 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya mengembangkan aplikasi **My Profile App** dari tugas minggu sebelumnya dengan menambahkan implementasi MVVM Pattern, fitur edit profile, dan fitur dark mode toggle.

## Fitur Aplikasi

1. Implementasi MVVM Pattern.
2. Membuat `ProfileViewModel` dengan StateFlow.
3. Data class `ProfileUiState`.
4. Form edit profile untuk mengubah nama dan bio.
5. State hoisting untuk TextField.
6. Tombol save untuk menyimpan perubahan melalui ViewModel.
7. Switch untuk mengaktifkan dan menonaktifkan dark mode.
8. State dark mode disimpan di ViewModel.

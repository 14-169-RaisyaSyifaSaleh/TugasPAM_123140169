## Deskripsi Tugas

Repository ini dibuat untuk memenuhi Tugas Praktikum Minggu 3 mata kuliah Pengembangan Aplikasi Mobile.

Pada tugas ini, saya membuat aplikasi **My Profile App** menggunakan Kotlin dan Jetpack Compose. Aplikasi ini menampilkan halaman profil sederhana yang berisi foto profil, nama, biodata singkat, serta informasi kontak.

## Fitur Aplikasi

1. Menampilkan header dengan foto profil dan nama.
2. Menampilkan biodata singkat.
3. Menampilkan informasi email, phone, dan location.
4. Menggunakan reusable composable function.
5. Menggunakan komponen UI seperti Column, Row, Box, Card, Text, Button, dan Image/Icon.

## Teknologi yang Digunakan

- Kotlin
- Jetpack Compose
- Android Studio

## Cara Menjalankan Aplikasi

1. Clone repository berikut:

   ```bash
   git clone https://github.com/14-169-RaisyaSyifaSaleh/TugasPAM_123140169.git
   ```
2. Buka project menggunakan Android Studio.
3. Jalankan aplikasi pada Android Emulator atau perangkat Android.
